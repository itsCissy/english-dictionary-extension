# 📚 English Dictionary - 划词词典插件

划词查看英英词典解释的浏览器扩展，支持生词本和发音功能。

## ✨ 功能

- 🔍 **划词即查**：选中网页上的英文单词，自动显示英英词典解释
- 🔊 **单词发音**：点击发音按钮听标准发音
- 📚 **生词本**：保存单词到生词本，支持标记掌握、导出
- 📖 **完整释义**：显示词性、定义、例句
- 📱 **侧边栏**：右侧面板实时查看生词本
- 🎨 **美观界面**：简洁设计，流畅动画
- ⚡ **快速响应**：使用 Free Dictionary API，无需等待

## 🚀 如何安装

### Chrome 浏览器

1. 打开 Chrome，在地址栏输入: `chrome://extensions/`
2. 右上角打开 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择这个文件夹: `~/dict-extension`
5. 完成！

### Firefox 浏览器

1. 打开 Firefox，在地址栏输入: `about:debugging#/runtime/this-firefox`
2. 点击 **"临时载入附加组件"**
3. 选择这个文件夹中的 `manifest.json`
4. 完成！

## 📖 如何使用

### 划词查词

1. 在任意网页上**用鼠标选中一个英文单词**
2. 稍等片刻（约 0.3 秒），词典弹窗会自动出现
3. 点击 **🔊** 按钮听发音
4. 点击 **📚 保存** 按钮添加到生词本

### 生词本

**方式一：侧边栏（推荐）**
- 右键点击浏览器工具栏的扩展图标
- 选择 **"打开侧边栏"**
- 侧边栏会固定在浏览器右侧

**方式二：选项页面**
- 右键点击扩展图标 → **选项**
- 或在 `chrome://extensions/` 页面点击 **详细信息** → **扩展程序选项**

### 生词本功能

| 功能 | 说明 |
|------|------|
| 🔍 搜索 | 输入关键词搜索单词或定义 |
| ✓ 掌握 | 标记已掌握的单词 |
| 🗑️ 删除 | 删除不需要的单词 |
| 📤 导出 | 导出为 CSV 文件（可用 Excel 打开） |
| 🗑️ 清空 | 清空所有单词 |

## 📁 文件说明

| 文件 | 作用 |
|------|------|
| `manifest.json` | 扩展配置文件 |
| `content.js` | 核心功能：监听选词、调用 API、显示弹窗 |
| `content.css` | 弹窗样式 |
| `options.html/js/css` | 生词本选项页面 |
| `sidepanel.html/js/css` | 侧边栏生词本 |

## 🔧 API 来源

使用 [Free Dictionary API](https://dictionaryapi.dev/) - 免费、无需注册、无限制。

## 🎯 快捷操作

| 操作 | 效果 |
|------|------|
| 选中单词 | 自动显示词典（0.3秒延迟） |
| 点击外部 | 关闭弹窗 |
| 按 Esc | 关闭弹窗 |
| 右键扩展图标 | 打开侧边栏/选项 |

## 🐛 已知问题

- 不支持在 PDF 文件中使用
- 某些特殊网站可能因为安全策略无法注入脚本
- Firefox 不支持侧边栏功能（请使用选项页面）
