# 📚 English Dictionary - 划词词典插件

划词查看英英词典解释的浏览器扩展。

## ✨ 功能

- 🔍 **划词即查**：选中网页上的英文单词，自动显示英英词典解释
- 📖 **完整释义**：显示词性、定义、例句
- 🎨 **美观界面**：渐变色设计，流畅动画
- ⚡ **快速响应**：使用 Free Dictionary API，无需等待

## 🚀 如何安装

### Chrome 浏览器

1. 打开 Chrome，在地址栏输入: `chrome://extensions/`
2. 右上角打开 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择这个文件夹: `~/dict-extension`
5. 完成！

### Firefox 浏览器

1. 打开 Firefox，在地址栏输入: `about:debugging#/runtime/this-firefox`
2. 点击 **"临时载入附加组件"**
3. 选择这个文件夹中的 `manifest.json`
4. 完成！

## 📖 如何使用

1. 在任意网页上**用鼠标选中一个英文单词**
2. 稍等片刻（约 0.3 秒），词典弹窗会自动出现
3. 点击右上角 **×** 或按 **Esc** 键关闭弹窗
4. 点击页面其他地方也会自动关闭弹窗

## 📁 文件说明

| 文件 | 作用 |
|------|------|
| `manifest.json` | 扩展配置文件 |
| `content.js` | 核心功能：监听选词、调用 API、显示弹窗 |
| `content.css` | 弹窗样式 |

## 🔧 API 来源

使用 [Free Dictionary API](https://dictionaryapi.dev/) - 免费、无需注册、无限制。
